v1.0.1 (18/04/26)# Заключение по настройке серверной инфраструктуры Urban

Успешно завершена настройка распределенной серверной части проекта на базе Ubuntu Server. Инфраструктура развернута с соблюдением принципов модульности и безопасности (согласно §1.1, §3.1 и §6.3 GUIDELINES).

## Итоговая архитектура
1.  **VM "DB" (192.168.1.170):** Backend-ядро на PocketBase (v0.23.2). Хранит данные (SQLite) и управляет логикой коллекций. Слушает внутренний порт 8090.
2.  **VM "APP" (192.168.1.171):** Обратный прокси-сервер (Nginx). Служит единой точкой входа.
    *   Настроен SSL-сертификат (Let's Encrypt) для домена `https://urban42.online`.
    *   Реализовано проксирование запросов: `/api/` -> PocketBase API, `/_/` -> PocketBase Admin UI.
3.  **VM "MAP" (192.168.1.172):** Резервный узел для развертывания собственного сервера тайлов (OSM) в рамках отказа от внешних SDK.

## Сетевой статус
*   **Белый IP:** 109.174.3.219
*   **Домен:** urban42.online
*   **Безопасность:** Весь внешний трафик идет по порту 443 (HTTPS). Прямой доступ к портам БД извне закрыт.

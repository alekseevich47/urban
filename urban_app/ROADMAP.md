# 🏙️ Urban: Cyberpunk App Roadmap

Этот документ служит «живой памятью» проекта. Здесь фиксируются все важные изменения, исправленные ошибки и план развития функционала.

---

## ✅ Выполнено (Done)

- [x] **Окружение (Environment):**
  - Настроен локальный Gradle 8.14.4 (для обхода сетевых ограничений).
  - Проект адаптирован для запуска в **Chrome (Web)** как основной среде тестирования.
  - Настроены неоновые цвета UrbanTheme (Фиолетовый #6C63FF, Голубой #00D9FF, Розовый #FF2E6D).

- [x] **Архитектура данных & Backend:**
  - Создана модель `EntertainmentVenue` с методами `fromJson/toJson`.
  - Внедрен **Repository Pattern** (интерфейс `VenueRepository`, реализации `Mock`, `Supabase` и **PocketBase**).
  - Настроен **Provider** для управления состоянием данных (VenueProvider).
  - Интеграция **PocketBase**: проект переведен на суверенный стек для независимости от зарубежных сервисов.

- [x] **Авторизация (Authentication):**
  - Реализован **LoginScreen** в стиле Cyberpunk/Glassmorphism.
  - Подключен **PocketBase Auth** (с поддержкой локального хранения сессии).
  - Система отслеживания состояния сессии (автоматический переход Login -> Home).

- [x] **Карта (MapScreen):**
  - Интеграция `flutter_map` с темной темой (CartoDB Dark).
  - Кастомные неоновые маркеры (Клубы, Бары, Рестораны).
  - Всплывающая карточка заведения (Details Panel) при клике на маркер.
  - Интерактивный поиск и фильтрация по категориям.
  - **Кнопка "Все категории"**: Реализована сетка (GridView) в модальном окне для навигации по всем типам заведений.
  - Временный Bypass авторизации для ускоренного тестирования UI.

- [x] **Лента (FeedScreen):**
  - Реализован список событий с карточками в стиле Glassmorphism.
  - Добавлены элементы взаимодействия: Лайки, Чат, Бронирование.
  - Переход на реактивное обновление данных через `VenueProvider`.

## 🛠️ Исправленные ошибки (Bug Fixes)

1. **Retina Mode Placeholder:** Исправлено предупреждение `flutter_map` путем добавления `{r}` в URL и `retinaMode: true`.
2. **Image 404:** Заменена битая ссылка на изображение Unsplash в моковых данных.
3. **EdgeInsets Error:** Исправлена опечатка `EdgeInsets.bottom` -> `EdgeInsets.only(bottom: ...)`.
4. **Git Push Failure:** Добавлено исключение в `.gitignore` для тяжелого архива Gradle (214МБ).
5. **Main.dart Cleanup:** Устранено дублирование функций `main` и классов `MainApp` после интеграции провайдеров.

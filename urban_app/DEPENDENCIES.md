## 🔄 ПРОЦЕДУРА ОБНОВЛЕНИЯ

1. Проверить `flutter pub outdated`
2. Прочитать CHANGELOG пакета
3. Обновить версию в `DEPENDENCIES.md`
4. Обновить версию во всех `pubspec.yaml`
5. Запустить тесты
6. Запустить сборку
7. Закоммитить изменения

❌ **ЗАПРЕЩЕНО:**
- Без явного разрешения оператора (человека) менять версии и модули
- Обновлять версии без тестирования
- Использовать разные версии в разных модулях

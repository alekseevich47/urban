## 1. ОЧИСТКА КЭША

### 1.1 Ручная очистка (Приоритет: ВЫСОКИЙ)

**Версия:** v1.0.0  
**Статус:** ✅ В разработке

**Описание:**
Пользователь может вручную очистить кэш в настройках приложения.

**Экран настроек:**
Настройки → Хранилище → Очистить кэш

**Что очищается:**
- [ ] Кэш изображений (cached_network_image)
- [ ] Кэш заведений (Hive)
- [ ] Кэш карты (Yandex Maps / Mapkit)
- [ ] Временные файлы

**Что НЕ очищается:**
- [ ] Токены авторизации
- [ ] Настройки пользователя
- [ ] Избранное
- [ ] Сохранённые профили

**Пример реализации:**
```dart
class CacheManager {
  Future<void> clearCache() async {
    // Очистка кэша изображений
    await DefaultCacheManager().emptyCache();
    
    // Очистка Hive
    await Hive.box('places').clear();
    await Hive.box('map_cache').clear();
    
    // Очистка временных файлов
    await Directory('/storage/emulated/0/Android/data/com.urban.app/cache')
        .delete(recursive: true);
    
    logger.i('Cache cleared successfully');
  }
  
  Future<int> getCacheSize() async {
    // Подсчёт размера кэша в МБ
    // Вернуть для отображения в настройках
  }
}
```

## 1.2 Автоматическая очистка (Приоритет: СРЕДНИЙ)
Версия: v2.0.0

Статус: ⏳ Планирование

Описание:
Предложение очистить кэш при достижении лимита.

Триггеры:

| Событие           | Действие                             |
|-------------------|--------------------------------------|
| Кэш > 500 МБ      | Показать уведомление "Очистить кэш?" |
| Кэш > 1 ГБ        | Показать предупреждение "Мало места" |
| Свободно < 100 МБ | Предложить очистку                   |

Пример:
```dart
class CacheMonitor {
  static const int warningThreshold = 500 * 1024 * 1024; // 500 МБ  static const int criticalThreshold = 1024 * 1024 * 1024; // 1 ГБ
  
  Future<void> checkCacheSize() async {
    final size = await CacheManager().getCacheSize();
    
    if (size >= criticalThreshold) {
      showCriticalCacheWarning();
    } else if (size >= warningThreshold) {
      showCacheCleanupSuggestion();
    }
  }
}
```

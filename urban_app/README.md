# Urban App - Прототип платформы для поиска развлечений

## 📱 Описание проекта

**Urban** — это информационная платформа для поиска ЛЮБЫХ развлечений в ЛЮБОМ городе. От малобюджетных активностей до мега-концертов.

## 🎯 Ключевые возможности

### Для пользователей:
- **Интерактивная карта** с фильтрацией по категориям развлечений
- **Лента контента** в двух режимах: список и инновационная "сфера"
- **Автоматические отзывы** — фото с места автоматически становятся отзывом
- **Социальные функции**: друзья, чаты, планирование уик-эндов
- **Пользовательские ивенты**: создание своих встреч и сходок
- **Геймификация**: уровни, очки, достижения
- **Свидания вслепую**: подбор партнера по параметрам

### Для предпринимателей:
- **Бесплатный бизнес-аккаунт** с базовым функционалом
- **Шаблоны для разных типов заведений** (кинотеатры, прокаты, бары и т.д.)
- **Управление бронированиями** и ценами
- **Платные опции продвижения**: премиум-размещение, реклама

## 🚀 Технологический стек

### Для MVP:
- **Flutter** — кроссплатформенная разработка (iOS + Android)
- **Provider** — управление состоянием
- **flutter_map** + **Mapbox** — кастомные карты
- **Firebase/Supabase** — бэкенд (аутентификация, БД, хранилище)
- **GoRouter** — навигация

### Для масштабирования:
- **GraphQL** — эффективные запросы данных
- **WebSocket** — realtime чаты и уведомления
- **Redis** — кэширование
- **Kubernetes** — оркестрация контейнеров

## 📋 Структура данных

### Основные сущности:

1. **EntertainmentVenue** — заведение/развлечение
   - Категория, координаты, рейтинг, отзывы
   - Специфичные фичи для каждого типа

2. **UserReview** — отзыв пользователя
   - Автоматически создаётся при публикации фото
   - Фото, текст, лайки

3. **UserEvent** — пользовательское мероприятие
   - Дата, место, лимит участников
   - Теги, приватность

4. **UserProfile** — профиль пользователя
   - Интересы, посещённые места, друзья
   - Уровень, очки, достижения

## 📈 Стратегия привлечения

### Предприниматели (B2B):
1. **Ручной охват** первых 100 заведений в пилотном городе
2. **Тариф "Основатель"** — пожизненная бесплатность для ранних
3. **Бесплатное заполнение** профиля силами команды
4. **Партнёрство** с ассоциациями бизнеса

### Пользователи (B2C):
1. **Партизанский маркетинг**: стикеры с QR в заведениях
2. **Реферальная программа**: бонусы за приглашение друзей
3. **Контент-маркетинг**: челленджи в соцсетях
4. **Коллаборации** с локальными блогерами
5. **Ивенты-запуски** в партнёрских заведениях

## ⚠️ Риски и решения

| Риск | Решение |
|------|---------|
| Проблема "курицы и яйца" | Запуск в одном городе, ручной онбординг |
| Высокая конкуренция | Упор на уникальные фичи (сфера, авто-отзывы) |
| Сложность модерации | AI-модерация + сообщество репортеров |
| Технические сложности | Поэтапная разработка, начиная с MVP |

## 🎁 Уникальные преимущества

1. **Авто-отзывы**: фото = отзыв (решает проблему лени)
2. **Лента-сфера**: инновационный формат просмотра
3. **Универсальность**: все развлечения в одном месте
4. **Геймификация**: уровни, очки, достижения
5. **Ивенты от пользователей**: не только официальный контент

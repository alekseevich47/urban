# 🏗️ Core Module

## 📝 Описание
Ядро приложения Urban. Содержит базовую инфраструктуру, общие утилиты, сетевой слой и централизованную дизайн-систему. Модуль является фундаментом для всех `feature-` модулей и не должен зависеть от них.

## 📂 Структура
- **`ui/`**: Дизайн-система Urban. Содержит `urban_theme.dart` (цвета, шрифты, стили, сетка 8px).
- **`network/`**: Инфраструктура для работы с сетью. Здесь находится `PocketBaseService` для взаимодействия с бэкендом.
- **`utils/`**: Общие вспомогательные классы. Например, `UrbanLogger` (§1.5) для безопасного логирования.
- **`di/`**: (В процессе) Настройка Dependency Injection для всего приложения.

## 🛠 Основные правила (§1)
- Запрещено использовать `print()`, только `UrbanLogger`.
- Все UI-константы должны браться из `UrbanTheme`.
- Модуль `core` не может импортировать ничего из `features/`.

## 📦 Зависимости
- `pocketbase`: ^0.23.2
- `dio`: ^5.9.2
- `logger`: ^2.4.0

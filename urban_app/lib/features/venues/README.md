# 🍹 Venues Feature Module (feature-venues)

## 📝 Описание
Модуль для работы с заведениями (клубы, бары, рестораны). Позволяет просматривать список (ленту) и карту заведений, фильтровать их по категориям и искать по названию.

## 📂 Структура (src/)
- **`domain/`**: Сущность `EntertainmentVenue`, интерфейс репозитория и юзкейсы для бизнес-логики.
- **`data/`**: Реализация репозитория, работающая с `PocketBaseService`.
- **`presentation/`**: 
  - `bloc/`: Управление состоянием (загрузка, фильтрация).
  - `pages/`: Экран ленты (`FeedScreen`) и экран карты (`MapScreen`).
  - `widgets/`: Специфичные для заведений компоненты (карточки, фильтры).

## 🔗 Зависимости
- `:core` (UI, Network, Utils)
- `:feature-map` (для отображения заведений на карте)

## 🛠 Особенности реализации
- Использует универсальный компонент `UrbanMapView` из модуля `:feature-map`.
- Масштабируемая фильтрация через `VenueCategory`.
- Соответствует §3.1 (Clean Architecture + BLoC).

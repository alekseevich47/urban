# 🔐 Auth Feature Module (feature-auth)

## 📝 Описание
Модуль аутентификации и авторизации пользователей Urban. Управляет жизненным циклом сессии, регистрацией и входом через различные провайдеры (Email, Телефон, VK, Yandex).

## 📂 Структура (src/)
- **`domain/`**: Сущность `User`, интерфейс `AuthRepository` и юзкейсы (Login, Register, Logout).
- **`data/`**: Реализация репозитория, взаимодействующая с `PocketBaseService` и безопасным хранилищем для токенов.
- **`presentation/`**: 
  - `bloc/`: Управление состоянием авторизации (`AuthBloc`).
  - `pages/`: Экраны входа, регистрации и восстановления пароля.
  - `widgets/`: Поля ввода, кнопки социальных сетей.

## 🔗 Зависимости
- `:core` (Network, Utils, UI)

## 🛠 Безопасность (§6)
- Токены хранятся исключительно в `flutter_secure_storage`.
- Пароли никогда не логируются.
- Реализована проверка сложности пароля на стороне клиента.

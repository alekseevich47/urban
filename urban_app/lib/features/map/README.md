# 🗺️ Map Feature Module (feature-map)

## 📝 Описание
Универсальный модуль для работы с картами в приложении Urban. Инкапсулирует SDK Яндекс.Карт (`yandex_mapkit`) и предоставляет другим модулям простой интерфейс для отображения объектов, управления камерой и геолокацией.

## 📂 Структура (src/)
- **`domain/`**: Сущность `MapItem` (универсальный объект на карте).
- **`presentation/`**: 
  - `widgets/`: `UrbanMapView` — основной переиспользуемый виджет карты.

## 🔗 Зависимости
- `:core` (UI, Utils)
- `yandex_mapkit`: ^4.2.1
- `geolocator`: ^14.0.2

## 🛠 Особенности реализации
- **Изоляция SDK**: Другие фичи не знают о существовании `yandex_mapkit`, они работают только с `MapItem` и `UrbanMapView`.
- **Авто-геолокация**: Виджет сам запрашивает разрешения и центрирует камеру на пользователе при первом запуске.
- **Масштабируемость**: Подготовлен для внедрения кластеризации (§8.1 FUTURE_IMPLEMENTATIONS).

## ⚠️ Правила (§13.2)
- Модуль `map` **запрещено** импортировать в `core`.
- Модуль `map` не должен зависеть от конкретных бизнес-фич (например, `venues` или `events`). Используйте `data` поле в `MapItem` для передачи контекста.
